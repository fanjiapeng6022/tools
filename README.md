# tools
小程序集合

housetools.exe是一个中文版的房贷计算器，纯净绿色版本，下载后放在windows桌面点击打开后，即可使用，解决了大家买方时难以找到工具的难题，本工具无偿奉献给社会。
希望大家放心使用。操作说明：输入贷款金额、年利率、贷款期限，然后点击计算，即可得出每月还款金额和支付利息总额。方便大家贷款买房时权衡利弊。
